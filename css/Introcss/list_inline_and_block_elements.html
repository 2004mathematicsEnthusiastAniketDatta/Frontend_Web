inline and block elements

inline elements

Inline elements are those which are displayed inline with the text. They do not start on a new line and only take up as much width as necessary.

Examples of inline elements are <span> and <a>.

Block elements

Block elements are those which start on a new line and take up the full width available.

Examples of block elements are <div> and <h1>-<h6>.
