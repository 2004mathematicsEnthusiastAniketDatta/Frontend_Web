Global Attributes

Global attributes are attributes that can be used with any HTML element.

Here are some of the global attributes:

  • class
  • id
  • style
  • title
  • lang
  • dir
  • tabindex
  • accesskey
  • contenteditable
  • contextmenu
  • data-*
  • draggable
  • dropzone
  • hidden
  • spellcheck
  • translate
