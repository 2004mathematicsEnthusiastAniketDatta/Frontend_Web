classes and ids

Classes and ids are used to style elements in a web page. They are used to apply styles to elements without having to use inline styles.

Classes are used to style elements that are similar, while ids are used to style elements that are unique.Classes are required in Javascript in case of DOM manipulation also and react js requires className as well as id.

ids are used to style elements that are unique. Javascript targets unique elements for functionality with the help of ids. Follows document object model (DOM) and react js requires id as well as className. id is also required in styling specific elements in css. unique ids can be generated with ids.

iframes

Iframes are used to display a web page within a web page. They are used to create a seperate window within a web page.
